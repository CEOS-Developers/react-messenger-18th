# Key Features

- 모바일과 데스크탑 환경 모두 대응
- 모바일과 데스크탑의 작동 방식 차별화
- react router dom을 활용한 라우팅
- useState(local)와 zustand(global)를 활용한 상태 관리
- styled components를 활용한 스타일링
- 읽음 기능
- 메시지 좋아요 기능
- 홈화면에서 원하는 유저를 더블 클릭하면 main 유저 변경 기능
- 새로운 대화 시작 기능
- 홈, 채팅목록, 새로운 채팅 페이지에서 검색 기능
- 유저 프로필 변경 기능
